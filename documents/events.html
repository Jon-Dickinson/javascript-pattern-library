readystatechange

The readystatechange event is triggered when the readyState property of the Document object and the XMLHttpRequest object changes. document.readyState has three possible values: loading (the webpage is loading), interactive (the webpage has been parsed but external resources are still loading) and complete (the webpage and all external resources have ended Loading, load event is about to be triggered).

document.onreadystatechange = function () { if (document.readyState === 'interactive') { // ... } }

focus, blur

Since the focus and blur events do not bubble, they can only be triggered during the capture phase, so the third parameter of the addEventListener method needs to be set to true.

let formA = document.getElementById('formA'); formA.addEventListener ('focus', function(event) { event.target.style.background = 'cyan' ; }, true ); formA.addEventListener ('blur', function(event) { event.target.style.background = '' ; }, true );

event.type

The Event.type property returns a string indicating the type of event. The type of event is specified when the event is generated. This property is read-only.

var evt = new Event ('foo'); evt.type // "foo"

event.timeStamp

The Event.timeStamp property returns a millisecond timestamp indicating when the event occurred. It is calculated relative to the successful start of the page load.

var evt = new Event ('foo'); evt.timeStamp // 3683.6999999995896

event.preventDefault

The Event.preventDefault method cancels the browser's default behavior for the current event. For example, after clicking a link, the browser will jump to another page by default. After using this method, it will not jump to another page; for example, if you press the space bar, the page will scroll down a certain distance. Scrolled up. The premise of this method is that the cancelable property of the event object is true, if it is false, calling this method has no effect. Note that this method just cancels the default effect of the event on the current element and does not prevent the event from propagating. If you want to stop propagation, you can use the stopPropagation () or stopImmediatePropagation () method.

var cb1 = document.getElementById('checkboxA'); cb1.addEventListener ( 'click', function(e) { e.preventDefault (); }, false );
